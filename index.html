Tip: Navigate through the Hierarchy Using Breadcrumbs

To go to the parent of the inspected element it is very helpful to use breadcrumbs especially when

  • there are many children to the parent (which makes us scroll more to reach the parent)
  • hierarchy is too deep

Elements Tab: DOM Breakpoints

Hover on me to see a custom tooltip.

Break on subtree modifications

This allows us to pause the execution on any subtree modifications such as node adds/removals.

This tooltip here is added to document body when hovered on the element and removed from it immediately after we move away from the hovered element. In this situation, to inspect the tooltip or to know which code adds or removes the tooltip, we can set a subtree modifications breakpoint on the body.

More breaking options

In addition to subtree modifications we also have

  • attribute modifications - which allows us to break on attribute changes such as a class got added to the clnode removal, etc
  • node removal - which breaks when the current node on which the breakpoint is going to be removed from the DOM

We can see the list of all these breakpoints in "DOM Breakpoints" tab.
